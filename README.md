# Java Basics

A comprehensive collection of foundational Java programs and Object-Oriented Programming (OOP) concepts. 

## 📖 Overview
This repository serves as a practical learning resource for Java beginners. It contains various standalone `.java` files that demonstrate essential programming concepts, ranging from basic control flow and mathematical algorithms to data structures and core OOP principles.

## 📂 Topics Covered

The repository includes implementations across the following key areas:

### 1. Mathematical Operations & Algorithms
* `Factorial.java`
* `FibonacciSeries.java`
* `HCF.java` & `LCM.java`
* `PrimeCheck.java`
* `BinaryDecimalConversion.java`
* `DivisibilityCheck.java`
* `LeapYear.java`

### 2. Number & Array Manipulation
* `ReverseNumber.java`
* `SwapNumbers.java`
* `PalindromeCheck.java`
* `MaxofThree.java` & `SmallestInArray.java`
* `MergeArray.java`
* `MedianFinder.java`
* `AlternateNumbers.java` & `CountDigits.java`

### 3. Object-Oriented Programming (OOP)
* `Inheritance.java` & `inherit.java`
* `Interfaces.java`
* `constructor_overloading.java`
* `objects.java`
* `this_Static.java`
* `Hiding.java`
* `Person.java` & `User.java`

### 4. Data Structures
* `StackImplementation.java`
* `QueueImplementation.java`

### 5. Utilities & Real-World Logic
* **Conversions:** `Conversion.java`, `KiloMIles.java`, `MilesToKm.java`, `TemperatureConversion.java`
* **Systems:** `GradingSystem.java`, `AdmissionCheck.java`, `EmployeeRecruitment.java`, `CommissionDemo.java`, `Salary.java`
* **Misc:** `MultiplicationTable.java`, `PatternA.java`, `PatternB.java`
